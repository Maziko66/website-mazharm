> Literacy with Games — OYUN İLE OKUMA-YAZMA

Awarded Research Project TÜBİTAK 2209/A

This interactive learning platform was developed to facilitate independent Turkish literacy acquisition for 1st-grade students. The project aimed to transition traditional reading and writing exercises into an autonomous, self-paced, and engaging experience that eliminates the need for constant adult supervision.

CORE GAMEPLAY & LEARNING MODES

  • Car Driving — Navigate a 3D road by choosing paths corresponding to correct syllables or words heard via integrated audio cues.
  • Drink Making — Select and "pour" letters in precise sequence to spell target words correctly in a creative brewing environment.
  • Fishing — Complete spelling challenges by catching fish tagged with specific letters as they appear on screen.

TECHNICAL IMPLEMENTATION

  • Engine & Scripting — Unity Engine + C#, with Visual Studio Code and Git for version control.
  • 3D Design & Audio — Assets modeled in Blender; phonetic audio processed in Audacity for accurate Turkish pronunciation.
  • Dynamic Systems — Scriptable Objects for question pools and dictionaries, aligned with MEB (Ministry of National Education) curriculum standards.
  • Analytics Module — Local data logging tracking completion times, accuracy ratios, and student progress over time.

PROJECT IMPACT

By combining audio-visual feedback with 3D game interaction, the application accelerated literacy progress while fostering a genuine "love for learning". The modular design allows for future expansions and group-based learning in classroom settings.
